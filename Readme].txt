The model is for the detection of objects(in this case the number of peoples) from the already existing data.
The model use tenserflow pretrained model along with the colab enviroment.
To use this model You have to download the essential file provided and run the code in the colab enviroment.
The code will work only on colab and not on any other enviroment.
You can use the code on other enviroment by making some changes.
